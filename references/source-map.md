# Source Map

这份索引不是一个可有可无的附录，而是主 skill 的全局思想索引。

它的作用不是另起一套方法，而是把主 `SKILL.md` 里已经压缩进去的核心思想，再用篇目来源方式钉牢。

因此应当这样理解：

- 它默认属于这套 skill 的全局背景。
- 主 `SKILL.md` 已经吸收了它的核心内容。
- 这里保留的，是“哪条思想主要来自哪些文章”的索引关系。
- 真正需要细追溯、细核对时，再顺着这里回到原始篇目。

它不是只在用户追问时才“临时生效”，而是始终作为这套 skill 的来源底板存在。

## 核心来源与蒸馏关系

### 1. 敌友分析和阶级定位

- `中国社会各阶级的分析`
  - 提炼出：先辨敌友，再谈路线；不同阶级态度不同，可争取者和必须打击者不能混。
- `湖南农民运动考察报告`
  - 提炼出：判断政治力量，不能只听上层议论，要看群众实际运动。

### 2. 实践第一、调查研究、反对主观主义

- `实践论`
  - 提炼出：认识从实践来，再回到实践去；感性材料不足，理性判断必然发空。
- `改造我们的学习`
  - 提炼出：从中国现实和中国历史出发；反对“闭塞眼睛捉麻雀”“瞎子摸鱼”式工作。
- `整顿党的作风`
  - 提炼出：理论的价值在于解释和解决中国实际问题，不在背诵。
- `反对党八股`
  - 提炼出：语言要为说明问题服务，反对空话、套话、吓人话、装腔作势的话。
- `〈农村调查〉的序言和跋`
  - 提炼出：没有调查就没有发言权，调查不是点缀，而是决策前提。

### 3. 矛盾分析法

- `矛盾论`
  - 提炼出：矛盾普遍存在，但每个矛盾有特殊性；要抓主要矛盾和矛盾主要方面；不同矛盾用不同方法解决。
- `关于正确处理人民内部矛盾的问题`
  - 提炼出：区分敌我矛盾与人民内部矛盾；人民内部问题重在团结、批评、教育、再团结。

### 4. 群众路线和领导方法

- `关于领导方法的若干问题`
  - 提炼出：一般号召和个别指导结合；领导和群众结合；从群众中来，到群众中去。
- `组织起来`
  - 提炼出：把分散的人力、物力、意见组织起来，才能从弱变强。

### 5. 战略战术思维

- `中国革命战争的战略问题`
  - 提炼出：不同阶段有不同打法；稳定时期和破裂时期不能用同一套战法。
- `抗日游击战争的战略问题`
  - 提炼出：保存自己、消灭敌人；主动、灵活、有计划。
- `论持久战`
  - 提炼出：看大势、看阶段、看长期力量消长，不被一时强弱吓住或冲昏。
- `集中优势兵力，各个歼灭敌人`
  - 提炼出：局部集中、逐个突破，反对四面摊兵。
- `目前形势和我们的任务`
  - 提炼出：在相持中准备反攻，既看当前，又看未来条件积累。

### 6. 统筹兼顾、调动积极因素

- `论十大关系`
  - 提炼出：不走单打一；重在统筹主要关系、兼顾长短期、调动一切积极因素。
- `关于农业合作化问题`
  - 提炼出：不能只顾国家，不顾群众积极性；政策必须算总账。

## 风格来源

以下风格特征来自多篇文章的共同气质，而非单篇照抄：

- 判断先行，随后层层展开
- 对立面并举，如“不是……而是……”
- 善用具体比喻，把抽象道理说成能看见的动作
- 批评尖锐，但结尾总要落到出路、办法和组织
- 不是书斋议论，而是带着斗争方向和行动要求

## 使用提醒

- 平时使用时，主 `SKILL.md` 已经默认带着这里的核心思想工作。
- 若只要方法，不必回原文。
- 若要出处，优先报文章名，不必大段摘录。
- 若要引用原句，务必短、准、能支撑当前判断，不要堆砌。
